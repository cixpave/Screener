Connect to thinkorswim

thinkorswim is part of Charles Schwab. Live data comes through the Schwab Trader API using OAuth — your password is never entered here.

  1. Create a (free) app at developer.schwab.com and request the Market Data product.
  2. Set your app's callback URL to the value below.
  3. Paste your App Key here and click Authorize with Schwab.

Note: completing the token exchange requires a small local proxy (the Schwab API does not allow direct browser calls). Until then, Pulse runs on clearly-labelled demo data so you can learn the tools risk-free. Your App Key is stored only in this browser.
